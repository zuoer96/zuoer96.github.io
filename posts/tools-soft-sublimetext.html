发表于2024-11-13|更新于2024-11-13|编程
|总字数:219|阅读时长:1分钟|浏览量:

SublimeText3

官网下载: http://www.sublimetext.cn/3

指引: http://www.sublimetext.cn/support

破解

破解网站：https://hexed.it/

插件

::: tip Coder 💬

有些插件可能当时有效，后面过时了，看情况安装吧

:::

Html5

输入html5,敲击Tab键自动补全html5规范文档

Alignment

代码对齐，如写几个变量，选中这几行，Ctrl+Alt+A,就可以对齐了.

Emmet

一种快速编写html/css的方法

JQuery

支持JQuer规范的插件包

JSFormat

JS代码格式化插件。ctrl+alt+f就可以了。

还在烦恼每次打开都会弹出烦人的版本更新提示吗？

文件乱码插件

解决: 安装 ConvertToUTF8 插件

  • ctrl+shift+P install package
  • controlctrl+shift+P install package
  • ctrl+shift+P ConvertToUTF8
