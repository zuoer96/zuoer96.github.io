用来表示停顿和语气

- 句号 。

- 逗号 ，
- 顿号 、
- 叹号 ！
- 问号 ？
- 冒号 ：
- 分号 ；

句号（。）

```
1、用于句子末尾，主要表示句子的陈述语气。
（1）下周一我要会总公司开会。
（2）他把东西都收拾好了。
```

> 句号是汉语中唯一一个空心的标点符号

逗号（，）

```
1、表示一句话中间短暂的停顿，表示句子或话还没有说完。
（1）秋天来了，天气凉了。
（2）走吧，迟了就来不及了。
```

顿号（、）

```
1、表示溢短中并列词语之间的停顿；或者表示汉语序次与之后。
（1）我一到中国就游览了广州、上海和北京。
（2）想去旅游时要先问自己三个问题：
一、去哪里？
二、怎么去？
三、住哪里？
```

叹号（！）

```
1、用于句子末尾，表示强烈情绪。
表示感叹语气：（1）图书馆里多么安静啊！

表示祈使语气：（2）快走！

表示反问语气：（3）我怎么比得上他呀！
```

问号（？）

```
1、用于句末，表示疑问语气，可以表示疑问， 也可以表示反问。
（1）这本书多少钱？
（2）难道你知不道他是我弟弟吗？
```

冒号（：）

```
1、用于总说性或提示性词语后面，说：想：例如：
（1）他对我说：“不要紧张。”

2、需要说明的词语后面 
（2）展览会时间：2022年1月7号-2月8号。

3、用于书信中称呼语的后面 
（3）亲爱的爸爸：
	你好！
	。。。。。。
```

分号（；）

```
1、复句内部的并列关系分句之间的停顿
（1）如果你想去，你就去；如果你觉得麻烦，你就待在家里。
（2）上课时，同学们都会认真听讲；下课后，同学们也会认真复习。
```



### 记叙文及其写作规范

> 写事记叙文是指**围绕一件事**，把一件事情的**整个过程讲清楚**，并能从中**得出某种启示**的文章。

我们按写作对象的不同分类

#### 记叙文分类

- 写事记叙文
- 写人记叙文
- 描写景物记叙文

#### 五种场常见的写作表达方式

1、记叙：把人物的经历、言行和事务发展的过程交代出来。

（1）今天是教师节，**我在教室里送给老师一束花（一张卡片）。**

2、描写：具体形象的描绘人物、事物的外貌、形状等等。

（1）这是一束玫瑰花，**红红的花瓣，绿绿的叶子，花杆上有一些小刺。**
（2）卡片上画了一颗大树，**它有茂密的绿叶和又红又大的果子，树下有一个人正在给它浇水。在这棵树的旁边写着我感谢阿老师的话。**

3、抒情：抒发、表达感情。增强文章的感染力。

（1）老师，**谢谢你对我门的教导，**尤其是在教师节这一天。

4、议论：通过分析评论，表明自己的观点、态度、主张等。深化人物、事件、环境的意义，加深读者的认识。

（1）**我觉得**设立教师节非常必要。虽然我们不一定要在这一天才感谢老师，但专门设立这样的节日可以提醒学生们感恩，也提醒其他人关注教师工作。

5、说明：用简洁文字对客观事物进行解说。对有关的人物、事件、环境等做必要的交代，推动情节的发展.

（1）每年公历8月10日，是中国的教师节。

> 记叙文主要的手法是记叙，抒情、议论、说明是次要手法。

#### 记叙文写作的一般格式

1、标题要居中，如果是2个字，中间空一格。

2、正文和标题之间写作作者名字。

3、正文每个自然段开头空两格。

4、文稿中的标点符号一般不可放在句首。



## 02 写事记叙文

> 了解写事记叙文的基本要素、记叙文中的三种人称，能运用“要素”和“人称”完成一篇记叙文。

### 记叙的要素

在记叙一件事情的时候，我们需要把事情发送的时间、地点、人物和事件讲清楚，这就是记叙的四个要素。

为了让别人更清楚事件是什么，把事件分为起因、经过、结果三个过程。

> 记叙6要素：时间、地点、人物、起因、经过、结果

记叙的排列，从一般记叙角度，比如“打扫卫生 星期天上午 在家里 迈克”

可以排列为：“星期天上午，迈克在家里打扫卫生。”（时间+人物+地点+事情）

练习：

（1）妹妹 在书房里 晚上 看书 => 晚上，妹妹在书房里看书

（2）中午 在厨房 做饭 妈妈 => 中午，妈妈在厨房里做饭

（3）一头大象 在树下 睡觉 下雨天 => 下雨天，一头大象在树下睡觉

为了让事件更丰富，我们增加事件的起因。常见的处理顺序：原因+句子。

“打扫卫生 星期天上午 在家里 迈克 因为家里太脏了” => 因为家里太脏了，星期天上午迈克在家里打扫卫生。

为了让事件更丰富，我们增加事件的结果。常见的处理顺序：原因+句子+结果。

“打扫卫生 星期天上午 在家里 迈克 因为家里太脏了 最后家里焕然一新” => 因为家里太脏了，星期天上午迈克在家里打扫卫生，最后家里焕然一新。

### 记叙的人称

有三种：

- 第一人称为：“我” “我们”
- 第二人称为：“你” “你们”
- 第三人称为：“他” “他们”

> 一般用第三、第一人称写作比较常见，第二人称比较少见

第一人称为：“我” “我们”

```
1、用第一人称可以描写我的心理状态，表达我的感情，另一方面可以让文章亲切自然、真实可信。
（1）寒假的时候，我从中国回到印尼。在印尼的飞机场，我遇到了一件很倒霉的事，那就是我的行李不见了。事情是这样的，我下飞机拿到行李之后，就去洗手间。因为行李箱有点大，推到洗手间有点麻烦 balabala
```

第三人称为：“他” “他们”

```
1、作者站在第三者的角度，把别人经历的事情告诉读者。作者不仅能够描写这个人他的外貌、他的动作、他的语言，还可以介绍这个人他的心里在想什么。不受时间和空间的限制。从第三者的角度，对事情的来龙去脉（事情的原因和结果）介绍的更清楚。
2、用第三人称会给人一种客观、理性的感觉
（1）有一个人乏力的走出考场。他看上去脸色苍白，也没有力气，但是它的脸上有一丝笑意。
```

**可不可以只用第一人称写作呢？为什么一定要学习用第三人称来写作呢？  第一人称在有的场景下是有局限性的**

```
期末考试试题太难了。天气很热，我苦苦的思考，还是一道题也不会做。我的汗珠顺着耳边流下来。我的头发胀，觉得天旋地转—— => 这是"我"能知道的
咕咚一声我摔倒在地上，昏了过去。  
监考老师急忙跑过来把我扶起来。全班同学也都停住笔吃惊地朝我这里看。一时间教室里乱了起来。 => 这是"我"无法知道的
```

> 用第一人称“我”来记叙一件事情时，只能记叙“我”在当时所看到、听到、想到、了解到的事情，如果“我”当时看不到、听不到、想不到、了解不到的内容，就不能用第一人称来记叙。

改成第三人称：

```
期末考试试题太难了。天气很热，他苦苦的思考，还是一道题也不会做。他的汗珠顺着耳边流下来。他的头发胀，觉得天旋地转—— 
咕咚一声他摔倒在地上，昏了过去。  
监考老师急忙跑过来把他扶起来。全班同学也都停住笔吃惊地朝他这里看。一时间教室里乱了起来。
```



记叙中可以做人称的转换，第一人称->第三人称，第三人称->第一人称。



> 了解写事记叙文的三种记叙顺序、记叙文叙事过程中的“详略”，能综合运用不同的“记叙顺序”和记叙过程中的“详略得当”的方法完成一篇记叙文。

### 记叙的顺序

1、顺序：顺着时间从前往后说，按照事情发展的先后顺序来说。

```
（1）上周末，我balabala，这周，我balabala。。。。。。
```

2、倒序：将事情的结果或精彩的片段提前，再按照事情发展的顺序描写

这种写法交代了结果或精彩的地方，可以吸引人们继续读下去，追查事情的开头和原因。比较有趣。

```
（1）这周，我balabala，上周，我balabala
```

3、插叙：写一件事情时插入一段有关的内容，再接着写原来的事情

```
（1）上周末，我balabala。(我对上周这件事的评论/看法),我继续balabala,这周，我balabala。。。。。。
```

### 记叙的详略

详略要得当：跟主题密切相关的详细些；跟主题关系不大的简单写。(就需要明确自己的文字的主题)。

```
做饭
（1）我一开始还不会做饭，所以报了个培训班。（略写）
（2）我怎么做饭的 （详写）
（3）学会做饭我找到了女朋友 （略写）
```

如果要详写，就要知道详细的方法

1、把样子详写

（1）他做饭 -> 他做的是什么饭？ -> 把饭的样子写具体

（2）他做了一大盘五颜六色的炒饭。

2、把动作详写

（1）他做饭 -> 他是怎么做饭的？ -> 把做的动作写具体

（2）他认真地做饭。

3、把程序详写

（1）他认真地做饭。-> 他有多认真？ -> 把认真的程序写具体

（2）他认真地做饭，认真的有人敲门都没有注意到。

4、把时间、地点、起因、结果写出来、写具体。

> 一般是经过详写

### 记叙的线索

在一篇记叙文中记叙几件事的时候，可以把事情串在一起的**一样东西、一句话、一种活动**等，就叫线索。

通过不同的时间，描述的几件事中的共同的东西、反复出现的东西就是线索。

没有线索的文章，中心不明确，我们需要明确自己想写的是什么，绕后围绕线索展开描写。
